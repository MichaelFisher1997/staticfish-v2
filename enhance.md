# STATICFISH SITE REBUILD PLAN

## Objective
Rebuild the Staticfish website to reflect current and expanded service offerings, modernize the design, improve clarity of value, and position Staticfish as both a provider of small-business websites and a full-stack development partner capable of custom SaaS and tooling.

---

## OVERVIEW OF CHANGES
### Goals
- Redesign the website UI for a modern, professional brand presence.
- Expand service positioning beyond static sites to include CMS, web tools, extensions, and SaaS development.
- Improve clarity, navigation, trust, and conversion (portfolio, pricing, CTAs).
- Add SEO-friendly structure and pages.
- Add portfolio case studies and technical differentiators.

### Key Enhancements
- New hero section & branding language.
- Clear segmented service categories.
- New navigation structure.
- Updated pricing and optional support retainers.
- Testimonials, portfolio, and case studies.
- Contact & quote funnel improvements.
- SEO optimization.
- Modern UI layout for stronger first impression.
- Consistent typography, spacing, and responsive layout refinement.

## DESIGN / UI & BRANDING REBUILD
### Visual Goals
- Modern minimalist design
- Clean grid system, consistent spacing and typography
- Visual hierarchy that guides the user to CTA
- More images, screenshots, mockups rather than walls of text
- Logo and brand refresh optional
- Custom illustrations or iconography

### Layout Style Recommendations
- Full-width hero with large typography
- Service cards with icons and hover states
- Section-based layout, strong whitespace
- Sticky header navigation with CTA
- Dark/light theme possibility
- Responsive tested on all screen sizes

### UI Elements to Include
- Buttons with variants (primary, secondary, outline)
- Success / trust indicators (badges, logos, testimonials)
- Pricing table layout
- Contact forms using minimal friction
